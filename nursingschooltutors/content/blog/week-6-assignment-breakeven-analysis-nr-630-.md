## Part 1: Break-Even Analysis

### What Is a Break-Even Analysis?

A **break-even analysis** is a financial tool used to determine the point at which total revenue equals total costs — meaning the organization neither profits nor loses money. For nurse executives, break-even analysis is an essential skill for evaluating the financial viability of new programs, services, or units.

**The Break-Even Formula:**

```
Break-Even Volume = Fixed Costs ÷ (Revenue per Unit − Variable Cost per Unit)
```

Where:
- **Fixed Costs** = Costs that do not change with volume (rent, equipment, administrative salaries)
- **Revenue per Unit** = Reimbursement received per patient visit/procedure
- **Variable Cost per Unit** = Costs that change with each patient (supplies, medications, per-case staffing)

## Part 2: Worked Case Study Solution

### Outpatient Fertility Clinic Break-Even Analysis

**Scenario:** You and several business partners have decided to establish an outpatient fertility clinic. The following financial data has been provided:

**Given Information:**
- Fixed Costs: $750,000/year (equipment lease, administrative staff, facility overhead)
- Variable Cost per Visit: $200 (supplies, per-visit clinical staff)
- Revenue per Visit: $450 (insurance reimbursement + patient co-pay)
- Target Contribution Margin: $450 − $200 = $250 per visit

**Step-by-Step Calculation:**

**Step 1: Calculate Contribution Margin**
```
Contribution Margin = Revenue per Unit − Variable Cost per Unit
$450 − $200 = $250 per visit
```

**Step 2: Calculate Break-Even Volume**
```
Break-Even Volume = Fixed Costs ÷ Contribution Margin
$750,000 ÷ $250 = 3,000 visits per year
```

**Step 3: Interpret Results**
The clinic must achieve **3,000 patient visits per year** to break even. At fewer than 3,000 visits, the clinic operates at a loss. At more than 3,000 visits, the clinic generates profit. Monthly, this requires 3,000 ÷ 12 = **250 visits per month**.

## Part 3: Variance Analysis

### Sensitivity Analysis — What if Volume Changes?

Sensitivity analysis examines how changes in key variables affect the break-even point. Consider these scenarios:

- If fixed costs increase to $900,000: Break-even = $900,000 ÷ $250 = 3,600 visits/year
- If revenue per visit drops to $400: Contribution margin = $200; Break-even = 3,750 visits/year
- If variable costs increase to $250: Contribution margin = $200; Break-even = 3,750 visits/year
